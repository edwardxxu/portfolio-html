This project focused on mapping both internet access and median income by county; as well as the percent change between 2017 and 2018 in these two metrics. It found a strong positive correlation between median income and broadband access in 2018 (r = 0.742, p < 0.001), suggesting that higher-income counties tend to have higher rates of broadband access.

Methodology

The analysis incorporated the following key factors:

  • County-level household broadband access rate 2017-2018.
  • County-level median household income 2017-2018.
  • Maps were created in ArcGIS Pro, and correlation calculations were conducted in RStudio.

Key Findings

The analysis shows a strong positive correlation between median income and broadband access in 2018 (r = 0.742, p < 0.001). However, when examining year-over-year changes from 2017 to 2018, there is virtually no correlation between changes in median income and broadband access (r = 0.003, p = 0.919). This suggests that while higher-income counties tend to have better broadband access, short-term changes in income are not associated with changes in broadband access.

Tools Used

This project utilized Geographic Information Systems (GIS) software for spatial analysis and visualization, along with custom RStudio scripts for correlation calculations. The final maps were created using ArcGIS Pro, incorporating each variable to provide a visual overview of the data.
