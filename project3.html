This project demonstrates an automated workflow using FME (Feature Manipulation Engine) to collect and analyze real-time wind speed data from NOAA's network of marine buoys. The workflow connects to NOAA's public API, processes the incoming data stream, and calculates average wind speeds for specified time intervals.

Methodology

The workflow consists of several key components:

  • API Connection: Custom HTTP connector to access NOAA's real-time buoy data feed
  • Data Extraction: JSON parsing to extract relevant wind speed measurements
  • Data Processing: Filtering, averaging, and statistical analysis of wind speeds
  • Spatial Analysis: Mapping buoy locations and corresponding wind measurements
  • Automated Scheduling: Workflow runs at specified intervals to maintain current data

Key Features

The FME workflow provides several important capabilities for wind speed analysis:

  • Real-time data access from multiple NOAA buoy stations
  • Automated data cleaning and validation procedures
  • Configurable time intervals for wind speed averaging
  • Geographic filtering options for specific coastal regions
  • Export capabilities for various data formats and visualization tools

Tools Used

This project was developed using FME Desktop 2022, incorporating various transformers for API interaction, data processing, and spatial analysis. The workflow leverages FME's HTTP and JSON handling capabilities for seamless integration with NOAA's real-time data services.
